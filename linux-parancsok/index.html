Linuxos parancsok

Az alábbi táblázat a Linux-ban leggyakrabban használt parancsokat mutatja be.

Parancs Kapcsolók Példa használatra Leírás
ls -a -l ls -a Kilistázza az aktuális mappa tartalmát.
cd cd Dokumentumok Belépés másik mappába.
mkdir -p mkdir "Új mappa" Új mappa létrehozása.
rm -r rm valami.txt Fájl törlése.
rmdir rmdir "Új mappa" Mappa törlése.
cat cat valami.txt Fájl tartalmának kiírása.
pwd pwd Az éppen aktuális mappa kiíratása.
echo echo "Szöveg" Szöveg kiíratása.
head -n head valami.txt A fájl első 10 sorát írja ki.
tail -n tail valami.txt A fájl utolsó sorait írja ki.
wc -c -w wc valami.txt A fájlban található betűk, szavak, byteok számát adja vissza.
tree -L tree Mappaszerkezet megjelenítése.
mcedit mcedit valami.txt Fájlszerkesztő.
wget -q wget http://pelda.com Fájlok letöltése.
curl -O curl http://pelda.com Fájlok letöltése.
iconv -f -t iconv -f UTF-8 -t ISO-8859-1 valami.txt Fájlok konvertálása más karakterkészletre.
